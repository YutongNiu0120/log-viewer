# 项目记忆

`.project-memory/` 用于保存这个仓库的可复用上下文，减少每次进入项目时的重复扫描。

建议读取顺序：

1. `generated/project-summary.md`
2. `generated/development-playbook.md`
3. `generated/feature-index.md`
4. 按任务需要读取 `memory/current.md`、`memory/decisions.md`、`memory/milestones.md`

说明：

- `generated/` 下是可重建内容，仓库结构变化后可以刷新
- `memory/` 下是长期记忆，默认不覆盖
